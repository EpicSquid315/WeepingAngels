MAKE SURE YOU HAVE AUDIOMOD AND MODLOADER INSTALLED BEFORE INSTALLING THIS MOD.
MAKE SURE META_INF IS DELETED.

Installation:

1. Search for %appdata% and once found continue to your roaming folder and .minecraft folder.
2. Copy resources into the minecraft folder and paste, if a popup message comes up press the copy and override button.
3. Continue to the .minecraft/mods/ folder. If /mods does not exist create it.
4. Copy WeepingAngels.zip into .minecraft/mods/ 
5. Copy weepingAngels.PROPERTIES to .minecraft/mods/
6. Start up minecraft.exe and enjoy.

Configuration:

1. Open the weepingAngles.PROPERTIES with notepad++/notepad or similar application.
2. You may change the value after the '=' to a corresponding boolean (true or false), integer (whole number).
3. spawnrate must be a value between 1 and 100.
4. canTeleport must be either true or false. (This property tells the angel if it is able to teleport the player or not)
